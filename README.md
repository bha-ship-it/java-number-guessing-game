# Number Game

A simple Java console-based Number Guessing Game.

## Features
- Generates a random number between 1 and 100
- Takes user input
- Gives feedback:
  - Too High
  - Too Low
  - Correct
- Counts total attempts
- Continues until the correct number is guessed

## Technologies Used
- Java
- Random Class
- Scanner Class

## How to Run

```bash
javac NumberGame.java
java NumberGame
```
